# FinanceDAMT Frontend

App movil/web de finanzas personales construida con React Native y Expo.

## Stack

- React Native 0.81
- Expo SDK 54
- TypeScript
- React Navigation
- Axios
- TanStack Query

## Requisitos

- Node.js 20+ (recomendado LTS)
- Expo CLI (se instala con Expo)

## Inicio rapido

```powershell
 npm install
 npm run start
```

Comandos utiles:

```powershell
 npm run android
 npm run ios
 npm run web
```

## Configuracion de API

Define la URL base del API:

```powershell
 $env:EXPO_PUBLIC_API_BASE_URL="http://localhost:5000/api"
```
